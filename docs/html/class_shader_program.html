Detailed Description

Container for a GLSL shader program and its constituent shaders.

This is composed of two to three shaders (if using geometry shaders) compiled and linked into a "shader program".

This supports Vertex/Fragment Shaders and Vertex/Geometry/Fragment Shaders. Version 4 is not yet supported, and so Tessellation Control and Evaluator shaders are not presently available.

Precondition
A valid OpenGL context must be present to the program.
