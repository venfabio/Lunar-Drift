Detailed Description

Describes an attribute in a given vertex type for shader inputting.

Vertices can take many forms, yet describe similar data. Likewise, shaders can take the same data in different ways, depending on the index generated. Consequently, this class acts to map a given attribute as a "semantic", which is translated trivially into the shaders' input's name.

Precondition
A valid OpenGL context must be present to the program
