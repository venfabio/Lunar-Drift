Detailed Description

A singleton that aggregates and encaspsulates access to config files.

Most projects utilize .INI files to configure the program. They provide convenient storage places for configuration variables, and are both easily readable and modifiable by users from outside of the program. This singleton class allows these INI files to be conveniently handled and interfaced for easier use by programmers.

Note
"_engine" file is a virtual file for representing internal engine configuration states
