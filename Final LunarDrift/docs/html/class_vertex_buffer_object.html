Detailed Description

template<typename V>
class VertexBufferObject< V >

Specialized VertexBufferObject template, for handling generic vertices.

Template Parameters
VGeneric vertex parameter to use for the buffer

Whereas the VertexBufferObjectBase allows VBOs to be represented generically, this class acts as a template for providing vertices to the VBO. Consequently, when making a VBO, this class template should be used and parameterized with the vertex type used. When using VBOs in the abstract, consider passing around VertexBufferObjectBase objects instead.

Precondition
A valid OpenGL context must be present to the program
