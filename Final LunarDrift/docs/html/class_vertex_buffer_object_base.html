Detailed Description

Abstract base class for specialized VertexBufferObjects.

This class represents VBOs in the abstract, managing their representation in OpenGL in a way that's agnostic with regards to the data stored in the VBO themselves. In most cases, this class will only be used when passing around VBOs as an abstract concept; to work with VBOs, one should use the VertexBufferObject class template with the vertex type specified.

Precondition
A valid OpenGL context must be present to the program
